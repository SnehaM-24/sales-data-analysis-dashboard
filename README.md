# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing sales data to uncover key business insights such as revenue trends, profit performance, and regional sales distribution. An interactive dashboard was created to help stakeholders make data-driven decisions.

## 🛠 Tools Used
- Microsoft Excel (Data Cleaning, Pivot Tables)
- Power BI (Dashboard & Visualization)

## 📂 Dataset
Sales dataset containing order date, region, product category, sales, profit, and customer details.

## 📊 Steps Performed
1. Data cleaning and preprocessing in Excel
2. Created pivot tables for summary analysis
3. Imported cleaned data into Power BI
4. Designed interactive dashboard with filters and KPIs
5. Analyzed sales trends by region, category, and time
